## 📖 О проекте

Eva - это мощный русский голосовой ИИ-ассистент для умного дома и повседневных задач. Проект основан на Irene Voice Assistant от AlexBond и расширяет его функциональность современными возможностями.

---

## ✨ Возможности

### 🎯 Основные функции
- **Голосовое управление умным домом** через Home Assistant
- **Диалоговые навыки** с поддержкой циклов и условий
- **Визуальный конструктор навыков** для создания сценариев без программирования
- **Авто-создание навыков** через LLM
- **Таймеры и напоминания**

### 🤖 ИИ и распознавание речи
- **LLM интеграция**: Ollama, LM Studio, OpenAI
- **Распознавание речи**: Vosk
- **Озвучка ответов (TTS)**

### 🏠 Умный дом
- **Home Assistant** - полное управление умным домом- тестирование
- **Камеры Home Assistant** - разработке
- **Автоматизации с триггерами**:
  - По состоянию устройств
  - По времени
  - По текстовым командам

### 🔗 Интеграции

#### Умный дом и IoT
- Home Assistant
- MQTT (IoT устройства)

#### Медиа и контент
- Spotify / Navidrome
- RSS ленты / FreshRSS
- Погода (wttr.in, Open-Meteo)- замечен баг.
- Википедия- тест 
#### Мессенджеры
- Telegram бот
- Discord бот

### ⚙️ Технические возможности
- Docker деплой
- NTP синхронизация времени
- Веб-интерфейс

## 🚀 Быстрый старт

### Docker (рекомендуется)

```bash
git clone https://github.com/mrSaT13/Eva.git
cd Eva
docker-compose up -d
```

### Локальная установка

```bash
git clone https://github.com/mrSaT13/Eva.git
cd Eva
pip install -r requirements.txt
python -m eva -T web
```

После запуска откройте веб-интерфейс: **http://localhost:8086**
