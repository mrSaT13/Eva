# Eva Voice Assistant

Голосовой ИИ-ассистент для умного дома.

**Создатель:** mrSaT13  
**Репозиторий:** https://github.com/mrSaT13/Eva

## Быстрый старт

### Docker:
```bash
git clone https://github.com/mrSaT13/Eva.git
cd Eva
docker-compose up -d
```

### Локально:
```bash
git clone https://github.com/mrSaT13/Eva.git
cd Eva
pip install -r requirements.txt
python -m eva -T web
```

Откройте http://localhost:8086
